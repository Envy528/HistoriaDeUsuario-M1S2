# Inventory management program
This program let you add products to an inventory, show all the products in said inventory and calculate the inventory's total value and total products stored in it. This program uses python to work and runs in the terminal
## How do you execute it?
1. You need python installed in your system
2. You have to open a terminal or an IDE
3. Execute the python file "main.py"

## Logic used in the program
- Dictionary
- List
- Functions
- Conditionals
- While and For Loops
